
# WordSwipe 📚✨

WordSwipe — приложение для изучения английских слов в формате карточек и свайпов. Пользователь отмечает, знает ли слово, или отправляет его на повторение, постепенно формируя собственный список для изучения.

## Возможности

- 📖 Изучение английских слов по карточкам.
- 👆 Управление свайпами:
  - Свайп вправо — слово изучено.
  - Свайп влево — добавить слово на повторение.
- 🔊 Озвучивание произношения слов.
- 🗂 Разделение слов по темам.
- 📊 Статистика изучения:
  - количество изученных слов;
  - слова на повторении;
  - общее количество слов.
- 🔄 Режим повторения.
- 💾 Автоматическое сохранение прогресса.
- ♻️ Сброс прогресса обучения.

## Как работает приложение

1. Выберите тему или изучайте все слова сразу.
2. Прочитайте слово и его перевод.
3. Нажмите кнопку 🔊 для прослушивания произношения.
4. Сделайте свайп:
   - ➡️ вправо — «Знаю».
   - ⬅️ влево — «Повторить».
5. Отслеживайте свой прогресс в верхней панели.
